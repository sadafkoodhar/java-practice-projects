# java-practice-projects
Hands-on Java practice projects covering JSON parsing, data handling, and backend development basics.

Project 1: JSON Parsing Practice

This project is based on my practice of handling JSON responses in Java.

In this, I worked on different types of JSON structures such as:
- Simple JSON parsing
- Handling multiple records using loops
- Working with nested JSON objects
- Using List and Map to store data

I used dummy data to simulate backend responses and focused on improving my logic for data extraction and handling.
